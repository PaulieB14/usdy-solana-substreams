```bash
cd usdy-solana-substreams

# View dashboard
cat DASHBOARD.md
# or open in browser

```

### **🌐 Access Interactive Dashboard**
1. **GitHub Pages URL**: `https://paulieb14.github.io/usdy-solana-substreams/docs/dashboard.html`

